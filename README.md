2026 Modern Wall Calendar
A clean, stylish, and beginner-friendly web calendar built with pure HTML, CSS, and JavaScript. This project was designed to look like a modern printed wall calendar with a focus on clean typography and sharp UI elements.

Features
Complete 2026 Year: View all 12 months from January to December.

Interactive Navigation: Use "Next" and "Previous" buttons to flip through the year.

Modern Design: Features a mountain-themed header with a custom diagonal blue overlay.

Notes Section: A dedicated space on the side to simulate a real paper calendar.

Smart Highlighting: Automatically highlights "Today" if you are viewing the current date in 2026.

Built With
HTML5: For the structure and layout.

CSS3: For the styling, including CSS Grid for the calendar days and Clip-Path for the diagonal header shapes.

JavaScript: For the logic that calculates the correct day of the week for each month.
